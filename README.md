# IPL_Project
This project analyzes match data from the Indian Premier League 2022 season to understand team performance and match outcomes. The dataset includes information such as teams, venue, toss winner, scores, match winner, and player of the match.

Using Python and data analysis tools, the project explores patterns like the impact of toss decisions, winning margins, and top-performing players. Charts and visualizations are used to present insights clearly.

This project helps practice data cleaning, and data visualization, which are important skills for a Data Analyst. 📊
